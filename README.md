**Сделал: Максим Еремеев**

## Датасет

Вопросы турнира ОВСЧ (Открытый Всероссийский Синхронный Чемпионат) по спортивному «Что? Где? Когда?» — 21 сезон, 4579 вопросов с 2003 года. Датасет получен из дампа БД сайта gotquestions.online.

## Скачивание моделей

Необходимо создать папку `models/` и скачать модели:

**word2vec** (RusVectōrēs `tayga_upos_skipgram_300_2_2019` — skipgram, 300d, корпус Тайга ~5 млрд слов, ~600 MB):
```bash
mkdir models
wget https://vectors.nlpl.eu/repository/20/185.zip
unzip 185.zip -d models/tayga_w2v
mv models/tayga_w2v/model.bin models/tayga_upos_skipgram_300_2_2019.bin
```

**navec** (Natasha `navec_hudlit_v1`, художественные тексты, ~50 MB):
```bash
wget https://storage.yandexcloud.net/natasha-navec/packs/navec_hudlit_v1_12B_500K_300d_100q.tar -P models/
```

## Запуск

### CLI

```bash
# поиск через BM25 (модели не загружаются)
python main.py search --query "пушкин стихи"

# поиск через word2vec
python main.py search --query "барбос" --method word2vec --top-n 3

# поиск с исправлением опечаток
python main.py search --query "гиголб повест" --spellcheck

# рекомендации похожих вопросов
python main.py recommend --query "Черепаха — Зелёная. А Кот?" --method navec
```

Параметры команд `search` и `recommend`:

| Параметр | Сокращение | Описание | По умолчанию |
|----------|------------|----------|--------------|
| `--query` | `-q` | Текст запроса | обязателен |
| `--method` | `-m` | `bm25` / `word2vec` / `navec` | `bm25` |
| `--top-n` | `-n` | Количество результатов | `5` |
| `--spellcheck` | `-s` | Исправить опечатки | выключено |

### Веб-интерфейс

```bash
python main.py web
```

| Страница | Описание |

## Методы поиска

### BM25
Классический полнотекстовый поиск. Реализован через библиотеку [`rank_bm25`](https://github.com/dorianbrown/rank_bm25) (BM25Okapi). Скор каждого документа — взвешенная сумма IDF-весов слов запроса с учётом длины документа.

### word2vec
Семантический поиск. Модель: RusVectōrēs `tayga_upos_skipgram_300_2_2019` — skipgram, 300 измерений, обучена на корпусе Тайга (~5 млрд слов). Вектор документа/запроса — среднее векторов токенов; токены ищутся в формате `слово_POS` (формат RusVectores). Сходство — косинусное.

### navec
Семантический поиск. Модель: [Natasha navec](https://github.com/natasha/navec) `navec_hudlit_v1` — 300 измерений, обучена на художественных текстах (~12 млрд слов). Квантизированные векторы — быстрая загрузка (~1 сек) и низкое потребление памяти (~50 MB). Сходство — косинусное.

## Исправление опечаток

Реализовано в [spellcheck.py](spellcheck.py) классическим методом через расстояние Левенштейна. Словарь частот строится из самого корпуса при старте. Для каждого слова запроса перебираются кандидаты на расстоянии 1–2, из них выбирается наиболее частотное слово корпуса. Если кандидатов нет, слово остаётся без изменений.

## Рекомендации

Реализованы в [recommender.py](recommender.py): на вход принимается текст вопроса, выполняется поиск похожих документов через выбранный метод, исходный документ исключается из результатов. На странице результатов у каждого вопроса есть кнопка «Найти похожие».
